# Flask Todo App

A professional Todo Application built using Flask and SQLite.

## Features

- Add Tasks
- Update Tasks
- Delete Tasks
- Mark Task Complete
- Responsive UI
- SQLite Database

## Technologies Used

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/yourusername/flask-todo-app.git
cd flask-todo-app
pip install -r requirements.txt
python app.py
